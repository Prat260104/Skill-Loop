# Skill Loop: The "Learn & Build" Roadmap (30 Days)

Since you are learning *while* building, we will move at a comfortable pace. I will teach you the concepts (React Hooks, Spring Security, etc.) before we write the code.

**Strategy:** 2 Days of Learning + 1 Day of Coding per feature.

---

## Mod 1: Frontend Basics (React & UI)
**Goal:** Master React basics while building the UI Skeleton.

### 📅 Day 1-2: Learning React & Tailwind
*   **Study Goal:**
    *   What is React? (Components, JSX).
    *   What is Tailwind CSS? (Utility classes vs CSS).
    *   **Topic:** `useState` and `useEffect` hooks.
*   **Task:**
    *   [ ] Initialize the Project.
    *   [ ] Play with Tailwind classes (Change colors, layout).

### 📅 Day 3-4: The Landing Page (Framer Motion)
*   **Study Goal:**
    *   How animations work in web (Framer Motion basics).
    *   Responsive Design (Mobile vs Desktop).
*   **Task:**
    *   [x] Build the "Hero Section" (The first thing users see).
    *   [x] Make it responsive.

## Mod 2: Backend Foundations (Spring Boot)
**Goal:** Understand APIs and Database connections.

### 📅 Day 5-6: Spring Boot 101
*   **Study Goal:**
    *   What is a REST API? (GET, POST, PUT, DELETE).
    *   Controller vs Service vs Repository layer.
*   **Task:**
    *   [x] Create the Spring Boot project.
    *   [x] Write a simple "Hello World" API.

### 📅 Day 7-8: Database Design (PostgreSQL)
*   **Study Goal:**
    *   Relational Databases (Tables, Keys).
    *   JPA/Hibernate (How Java talks to SQL).
*   **Task:**
    *   [ ] Design the `User` table (Name, Email, Password).
    *   [x] Connect Spring Boot to PostgreSQL.

## Mod 3: Security & Auth (The Hard Part)
**Goal:** Mastering Login/Signup.

### 📅 Day 9-11: Authentication Theory
*   **Study Goal:**
    *   What is JWT (JSON Web Token)?
    *   What is OAuth2 (Login with Google)?
    *   Encryption (BCrypt).
*   **Task:**
    *   [ ] Implement Signup API (Save user to DB).
    *   [ ] Implement Login API (Generate Token).

### 📅 Day 12-14: Linking Frontend to Backend
*   **Study Goal:**
    *   `fetch` vs `axios`.
    *   Handling CORS errors (The most common error!).
*   **Task:**
    *   [ ] Create Login Form in React.
    *   [ ] Make it talk to your Spring Boot Backend.

---

## Mod 4: The Core Features (Skill Exchange)
**Goal:** Building the actual Startup logic.

### 📅 Day 15-18: Profile & Search

*   **Task:**
    *   [ ] Build "Edit Profile" page.
    *   [ ] Build "Search Mentor" page.

### 📅 Day 19-22: Sessions & Matching
*   **Study Goal:**
    *   State Management (Redux/Zustand) - complex data flow.
*   **Task:**
    *   [ ] Build "Request Session" button.
    *   [ ] Show list of "Upcoming Sessions".

**Goal:** Learning the high-paying skills.

    *   HTTP (One-way) vs WebSockets (Two-way communication).
*   **Task:**
    *   [ ] Build a simple Chat window explicitly for matched users.

### 📅 Day 29-30: Deployment & Review
*   **Study Goal:**
    *   Docker & Cloud Basics.
*   **Task:**
    *   [ ] Polish the UI.
